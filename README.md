# GolfStatRecorder

## Introduction
`GolfStatRecorder` is a comprehensive solution designed for local golf enthusiasts to keep track of their golf statistics. Built as my capstone project for a coding bootcamp, it incorporates a full stack of technologies to deliver a seamless user experience. Players can effortlessly register, manage their profiles, and record scorecards for every game they play.

## Key Features
- **Player Registration**: Users can create their player profile.
- **Profile Management**: Players can update their personal information.
- **Scorecard Tracking**: Users can create and manage scorecards after playing rounds of golf.
- **Statistics Overview**: Provides a dashboard for users to review their game statistics.

## Technologies Used
- **Java 1.8**: For constructing robust backend logic.
- **Jersey Framework**: To create RESTful web services in Java.
- **AngularJS**: For a responsive and interactive front-end experience.
- **MySQL/MariaDB**: For secure and reliable data persistence.
- **Bootstrap 4**: To craft a modern and mobile-friendly interface.
- **HTML**: Structuring the content of the web pages.


### Prerequisites
- JDK 1.8
- MySQL or MariaDB
- Node.js and npm (for AngularJS dependencies)
